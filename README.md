## Getting Started

### Prerequisites
- Python 3.x

### Setup
Clone the repository:

```bash
git clone https://github.com/VaibhavKalyan/CGS-410.git
cd CGS-410
```

If the project includes a `requirements.txt`, install dependencies via:

```bash
pip install -r requirements.txt
```
